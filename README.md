# AplicProgOO

Este repositório contem arquivos para estudo da matéria de Aplicações de Linguagem de Programação Orientada a Objetos.

## Como usar?

### Linux ou Mac

```bash
chmod +x gradlew
./gradlew build
java -jar build/libs/*.jar
```

### Windows com java > 8

```msdos
gradlew.bat
java -jar build\lib\*.jar
```
